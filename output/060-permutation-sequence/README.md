# [第k个排列][title]

## Description

给出集合 `[1,2,3,…, _n_ ]`，其所有元素共有  _n_! 种排列。

按大小顺序列出所有排列情况，并一一标记，当  _n_ = 3 时, 所有排列如下：

  1. `"123"`
  2. `"132"`
  3. `"213"`
  4. `"231"`
  5. `"312"`
  6. `"321"`

给定  _n_ 和  _k_ ，返回第  _k_  个排列。

**说明：**

  * 给定 _n_  的范围是 [1, 9]。
  * 给定 _k  _的范围是[1,   _n_!]。

**示例  1:**
            **输入:** n = 3, k = 3    **输出:** "213"    

**示例  2:**
            **输入:** n = 4, k = 9    **输出:** "2314"
