# [超级次方][title]

## Description

你的任务是计算  _a_ _b_  对 1337 取模， _a_ 是一个正整数， _b_ 是一个非常大的正整数且会以数组形式给出。

**示例 1:**
            **输入:** a = 2, b = [3]    **输出:** 8    

**示例  2:**
            **输入:** a = 2, b = [1,0]    **输出:** 1024

**Difficulty:** Medium
