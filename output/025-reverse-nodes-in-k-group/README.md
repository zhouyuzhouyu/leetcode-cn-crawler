# [K 个一组翻转链表][title]

## Description

给你一个链表，每  _k  _个节点一组进行翻转，请你返回翻转后的链表。

_k  _是一个正整数，它的值小于或等于链表的长度。

如果节点总数不是  _k  _的整数倍，那么请将最后剩余的节点保持原有顺序。



**示例：**

给你这个链表：`1->2->3->4->5`

当  _k  _= 2 时，应当返回: `2->1->4->3->5`

当  _k  _= 3 时，应当返回: `3->2->1->4->5`



**说明：**

  * 你的算法只能使用常数的额外空间。
  * **你不能只是单纯的改变节点内部的值** ，而是需要实际进行节点交换。
