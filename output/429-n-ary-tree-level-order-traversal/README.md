# [N叉树的层序遍历][title]

## Description

给定一个 N 叉树，返回其节点值的 _层序遍历_ 。 (即从左到右，逐层遍历)。

例如，给定一个 `3叉树` :



![](https://assets.leetcode-cn.com/aliyun-lc-
upload/uploads/2018/10/12/narytreeexample.png)



返回其层序遍历:
            [         [1],         [3,2,4],         [5,6]    ]    



**说明:**

  1. 树的深度不会超过 `1000`。
  2. 树的节点总数不会超过 `5000`。


**Tags:** Tree, Breadth-first Search

**Difficulty:** Medium
