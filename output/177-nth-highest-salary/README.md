# [第N高的薪水][title]

## Description

编写一个 SQL 查询，获取 `Employee` 表中第  _n  _高的薪水（Salary）。
            +----+--------+    | Id | Salary |    +----+--------+    | 1  | 100    |    | 2  | 200    |    | 3  | 300    |    +----+--------+    

例如上述 `Employee` 表， _n = 2  _时，应返回第二高的薪水 `200`。如果不存在第  _n  _高的薪水，那么查询应返回
`null`。
            +------------------------+    | getNthHighestSalary(2) |    +------------------------+    | 200                    |    +------------------------+
