# [去除重复字母][title]

## Description

给你一个仅包含小写字母的字符串，请你去除字符串中重复的字母，使得每个字母只出现一次。需保证返回结果的字典序最小（要求不能打乱其他字符的相对位置）。



**示例 1:**
            **输入:** "bcabc"    **输出:** "abc"    

**示例 2:**
            **输入:** "cbacdcbc"    **输出:** "acdb"



**注意：** 该题与 1081 <https://leetcode-cn.com/problems/smallest-subsequence-of-
distinct-characters> 相同


**Tags:** Stack, Greedy

**Difficulty:** Hard
