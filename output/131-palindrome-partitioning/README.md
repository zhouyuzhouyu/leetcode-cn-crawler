# [分割回文串][title]

## Description

给定一个字符串 _s_ ，将 _s_ 分割成一些子串，使每个子串都是回文串。

返回 _s_ 所有可能的分割方案。

**示例:**
            **输入:**  "aab"    **输出:**    [      ["aa","b"],      ["a","a","b"]    ]

**Difficulty:** Medium
