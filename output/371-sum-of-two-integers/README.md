# [两整数之和][title]

## Description

**不使用** 运算符 `+` 和 `-` ​​​​​​​，计算两整数 ​​​​​​​`a` 、`b` ​​​​​​​之和。

**示例 1:**
            **输入:** a = 1, b = 2    **输出:** 3    

**示例 2:**
            **输入:** a = -2, b = 3    **输出:** 1


**Tags:** Bit Manipulation

**Difficulty:** Easy
