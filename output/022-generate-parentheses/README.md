# [括号生成][title]

## Description

数字 _n_  代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。



**示例：**
            **输入：** n = 3    **输出：** [           "((()))",           "(()())",           "(())()",           "()(())",           "()()()"         ]    


**Tags:** String, Backtracking

**Difficulty:** Medium
