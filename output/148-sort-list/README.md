# [排序链表][title]

## Description

在  _O_ ( _n_  log  _n_ ) 时间复杂度和常数级空间复杂度下，对链表进行排序。

**示例 1:**
            **输入:** 4->2->1->3    **输出:** 1->2->3->4    

**示例 2:**
            **输入:** -1->5->3->4->0    **输出:** -1->0->3->4->5


**Tags:** Sort, Linked List

**Difficulty:** Medium
