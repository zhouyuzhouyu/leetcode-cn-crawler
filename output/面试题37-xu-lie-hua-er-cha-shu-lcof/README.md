# [序列化二叉树][title]

## Description

请实现两个函数，分别用来序列化和反序列化二叉树。

**示例:  **
            你可以将以下二叉树：            1       / \      2   3         / \        4   5        序列化为 "[1,2,3,null,null,4,5]"

注意：本题与主站 297 题相同：<https://leetcode-cn.com/problems/serialize-and-deserialize-
binary-tree/>
