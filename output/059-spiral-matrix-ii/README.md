# [螺旋矩阵 II][title]

## Description

给定一个正整数  _n_ ，生成一个包含 1 到  _n_ 2 所有元素，且元素按顺时针顺序螺旋排列的正方形矩阵。

**示例:**
            **输入:** 3    **输出:**    [     [ 1, 2, 3 ],     [ 8, 9, 4 ],     [ 7, 6, 5 ]    ]

**Difficulty:** Medium
